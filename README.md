# Focus Timer

A simple, single-page Pomodoro timer with preset durations and a custom minute picker. Everything runs locally in your browserâ€”no accounts or storage needed.

## Usage
1. Pick a preset or enter a custom minute value, then press **Set**.
2. Press **Start** to begin the countdown, **Pause** to stop temporarily, and **Reset** to return to the selected duration.
3. The outer ring fills as time elapses so you can gauge progress at a glance.
